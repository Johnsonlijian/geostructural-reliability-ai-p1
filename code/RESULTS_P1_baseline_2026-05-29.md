## 4. 诚实解读（这就是论文的论点）
1. **乐观性缺口真实存在**（ΔAUC 0.04–0.05）：随机划分高估了 ML 的跨条件泛化；按地震事件留出才是诚实数字。与你 CFST 论文的 ΔR² 同一签名，迁移到地震岩土域成立。
2. **三个现成 ML 在随机 CV 下就没超过 BI2014（0.923），震群分组下更低（0.84–0.87）**：在该库、诚实验证下，**黑箱 ML 不优于工程规范**。
3. → 直接**否定"换个 ML 刷 AUC"的套路**，论证你这条线的必要性：**机理基线打底 + 对基线的残差学习 + 地下水/有效应力隐藏变量 + 可靠性边界**，而非黑箱。这是一个可发表、可证伪、有锋芒的真实结论。

## 7. ①残差融合 + ②地下水控制（同日续跑，真实数据）

### ① 残差/机理-数据融合（震群分组，208 例/27 震）
| 模型 | AUC |
|---|---|
| M0 物理 BI2014（零样本，不拟合） | **0.923** |
| M1 校准物理（grouped） | 0.898 |
| M2 ML 仅特征（grouped） | 0.851 |
| M3 融合 特征+物理边际（grouped） | 0.858（随机 0.903，缺口 0.045） |

- 融合 vs 规范 **−0.065**（分组下**不超过**规范）；融合 vs 黑箱 +0.007。
- **硬结论**：在该单库、诚实震群分组下，**黑箱 ML 与机理-数据融合都没超过零样本 BI2014**，且乐观性缺口在融合后仍在（随机 0.903→分组 0.858）。→ 验证过的机理仍是最可靠基线；ML 表观增益主要来自随机划分。正面主张须建立在**跨库泛化 + 残差不确定性 + 地下水机理**，而非单库刷 AUC。产物 `cetin2018_residual_fusion.json`。

### ② 地下水/有效应力的机理控制
- 引擎二次自检：用深度/水位/容重重算 σ′v vs 数据集列 **r=0.989、MAE 6.6 psf**。
- **机理反事实**（把水位搬到 地表/实际/深处，过验证引擎）：

| 水位情景 | 中位 FS | 预测液化率(FS<1) |
|---|---|---|
| 地表 dw=0 | 0.640 | 0.726 |
| 实际 | 0.903 | 0.567 |
| 深/干 dw=z | 1.252 | 0.303 |

→ 地下水位是**触发的机理控制变量**：水位抬到地表，中位 FS 1.25→0.64、预测液化率 0.30→0.73；用错水位（按地表）灾害判定准确率 0.832→0.798。
- ML 落柱消融（震群分组）：去掉显式 σ′v+水位，AUC 0.856→0.857（≈0）。**相关性 AUC 对地下水不敏感**（总应力+深度可代偿），但**机理反事实证明其因果控制**——"相关重要性"掩盖"因果控制"，正是机理优先的论据。产物 `cetin2018_groundwater_ablation.json`。

## 8. ③ 跨源泛化 + 物理基线"划分不变性"（真实数据）

**数据获取说明（诚实记录）**：CPT 第二库 = Geyin & Maurer (2021)，DOI 10.17603/ds2-wftt-mv37 → DesignSafe **PRJ-3012**；其 .mat 在 DesignSafe 访问层之后（公开 API 只返回项目元数据、不给文件直链），**未能自动直拉**。改用 Cetin 库自带的 **`Reference`（原始来源研究）列**做 leave-one-source-out——回答同一科学问题（对未见来源是否泛化）的严格替代。

| 验证划分（严格度递增） | GBM AUC | logit AUC | 物理 BI2014（划分不变） |
|---|---|---|---|
| 随机 K 折 | 0.900 | 0.886 | **0.923** |
| 按地震分组（留整场地震） | 0.851 | 0.850 | **0.923** |
| 按来源研究分组（43 来源） | 0.880 | 0.876 | **0.923** |

n=208 / 27 次地震 / 43 个来源研究。产物 `cetin2018_source_grouped.json`。

**③ 核心结论**：
1. 三种划分下，零样本物理 BI2014（0.923）**从未被 ML 超过**（ML 最高仅在最乐观随机划分 0.90）。
2. **物理 AUC 划分不变**——它不拟合标签，按构造跨地震、跨来源泛化；ML 随划分退化、留整场地震时最低（0.85）。这就是"跨源/跨库泛化"的机理论据：**验证过的机理可迁移，拟合黑箱不可靠迁移**。
3. 随机较地震分组高 ~0.05 AUC——乐观性缺口在第二分组轴复现。
